# 🎯 Project Objective

The goal of this project is to predict the rating of a movie using various movie attributes.

Instead of manually estimating ratings, Machine Learning algorithms learn relationships between movie features and their corresponding ratings.

This project demonstrates an end-to-end Machine Learning workflow including:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Building
- Model Evaluation
- Prediction

---

# 📂 Dataset

The dataset contains information about movies including attributes such as:

- Movie Name
- Genre
- Director
- Duration
- Votes
- Year
- Other movie-related information
- Rating (Target Variable)

# 🤖 Machine Learning Models

The following regression models can be used for Movie Rating Prediction:

| Model | Purpose |
|--------|----------|
| Linear Regression | Baseline Model |
| Decision Tree Regressor | Non-linear Prediction |
| Random Forest Regressor | Ensemble Learning |
| Gradient Boosting Regressor | Improved Accuracy |

---

# 📈 Model Evaluation

The models were evaluated using regression metrics:

- Mean Absolute Error (MAE)

- Mean Squared Error (MSE)

- Root Mean Squared Error (RMSE)

- R² Score

Lower MAE/RMSE indicates better prediction performance.

Higher R² Score indicates better model fit.
